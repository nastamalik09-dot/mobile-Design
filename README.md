# Мои мобильные дизайны (Figma)

 О проекте
Практика интерфейсов популярных приложений с использованием Auto Layout, Components и Variants.

 Проекты

 1. Telegram Home Screen
**Описание:** Полный редизайн домашнего экрана Telegram.  
**Что отработано:** Auto Layout, Components, Variants, тёмная тема.  
**Ссылка:** [Открыть в Figma](вставь_ссылку)

 2. Tinkoff Mobile
**Описание:** Главный экран Тинькофф с карточками, балансом и быстрыми действиями.  
**Что отработано:** Сложные карточки, Auto Layout, иконки.  
**Ссылка:** [Открыть в Figma](вставь_ссылку)

 3. Spotify Home Screen
**Описание:** Главный экран Spotify с рекомендациями, плейлистами и мини-плеером.  
**Что отработано:** Горизонтальные карусели, карточки, плеер.  
**Ссылка:** [Открыть в Figma](вставь_ссылку)

---

**Технологии:** Figma, Auto Layout, Components, Variants

**Цель:** Развить навыки мобильного UI/UX дизайна.
